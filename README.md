## Что это

Сайт фотосалона «Юпитер» — тёмный, минималистичный, без фреймворков.  
Три страницы, каждая — самостоятельный инструмент.

---

## Страницы

```
index.html      →  Главная — услуги, цены, график, контакты
resume.html     →  Конструктор резюме онлайн
converter.html  →  Конвертер изображений и документов
```

---

## Стек

```
HTML · CSS · Vanilla JS
```

Без React, без сборки, без бэкенда — всё работает как статика на GitHub Pages.

---

## Инструменты

### 📄 Конструктор резюме
Заполни форму → загрузи фото → выбери цвет акцента → скачай PDF.  
Всё происходит в браузере, файл не уходит никуда.

### 🔄 Конвертер файлов
**Изображения** — JPG, PNG, WebP, BMP с настройкой качества и размера.  
**Документы** — DOCX, XLSX, CSV, TXT, HTML, JSON, PDF.  
Пакетная обработка, скачивание ZIP-архивом. Без серверов, без лимитов.
